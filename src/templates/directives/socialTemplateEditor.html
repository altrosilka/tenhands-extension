Редактор шаблона

Укажите в поле ниже какую именно информацию с сайта нужно использовать и в каком порядке

Можно использовать как простой текст, так и переменные. Переменные выделяются двойными фигурными скобками, например {{title}}

Доступны следующие переменные:

  • {{title}} - заголовок страницы
  • {{description}} - содержимое метатега description
  • {{url}} - ссылка на сайт
  • {{h1}}{{h2}}{{h3}} - текст из первых тегов h1, h2 или h3
  • {{url_clear_hash}} - ссылка на сайт без якоря (текст после символа #)
  • {{url_clear_search}} - ссылка на сайт без текста после символа ?

Значения переменных на текущей странице (для примера):

title

description

url

h1

h2

h3
